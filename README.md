
# Business Performance KPI Dashboard

## Project Overview
This project presents a Business Performance KPI Dashboard created using Power BI. It analyzes sales, profit, customer segments, regional performance, and seasonal trends using the SuperStore Sales Dataset.

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Profit Margin KPI
- Total Customers KPI
- Sales by Region (Bar Chart)
- Profit by Category (Column Chart)
- Monthly Sales Trend (Line Chart)
- Top Customers Table
- Sales by Segment (Pie Chart)
- Interactive Filters/Slicers

## Business Analysis
The dashboard helps answer key business questions such as:
- Which region generates the highest revenue?
- Which category has the highest profit margin?
- Are discounts reducing profitability?
- Which customer segment is most valuable?
- Is there a seasonal sales trend?

## Tools Used
- Power BI
- Microsoft Excel
- SQL (for data analysis)
- SuperStore Sales Dataset

## Key Insights
- Central region generated the highest sales revenue.
- Technology category achieved the highest profit margin.
- Discounts above 20% reduced profitability.
- Consumer segment contributed the highest total sales.
- Q4 showed the strongest seasonal sales performance.
